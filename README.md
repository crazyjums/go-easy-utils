## Introduction

Quick toolbox for common data processing developed by Go language

### Installation

Go version >= 1.16 (recommend)

```bash
go get -u github.com/jefferyjob/go-easy-utils

import (
	"github.com/jefferyjob/go-easy-utils"
)
```
